Expand description

Contains flags for the focus.

This struct is used as part of the widget state.

See HasFocusFlag, validate! and also on_gained!, on_lost!.

Fields§

§tag: Cell<u16>

A unique tag within one focus-cycle. It is set when the focus cycle is created. See Focus::focus

§focus: Cell<bool>

Focus.

§gained: Cell<bool>

This widget just gained the focus.

See on_gained!, validate!

It is reset at the beginning of all handle_xxx() calls.

§lost: Cell<bool>

This widget just lost the focus.

See on_lost!, validate!

It is reset at the beginning of all handle_xxx() calls.
